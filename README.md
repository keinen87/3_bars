# The nearest bars

The script helps to find biggest, smallest and closest bar

# Quickstart
Launch on Linux(Python 3.5) or Windows as simple

```#!bash

$ python bars.py <path_to_file>
# For example, python bars.py '/home/user/downloads/bars.json'
```
